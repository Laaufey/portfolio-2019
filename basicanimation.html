Projectpool 1

Goal

Project Pool is a pool from KEA full of html, css and javascript projects to choose from. The rules are 1. HTML, CSS, JS are equally important. It should look good. It should feel good. It should work. 2. A new partner every day. 3. A new project every day. Don't work on the same with two partners. But show your work, get inspired.

Focus

In teams of two, we chose to make a game called the fly game. When we made the game we chose to use my adorable illustration drawing of my cat, found some cat sounds, and called the game Catch the Cat. After making the game I attached it to my page I made the week before with the javascript animations. I added the button "game" where it can be played.

Onboarding

Goal

Creating an interactive, animated, web based onboarding sequence that through gamification explains an important aspect about moving to Denmark to be a student on KEA multimedia design. In a team of four people, we created a game called All in One. The game is to help new and future international students in Denmark to know how to survive the necessary paperworks when you arrive to the country.

Focus

The assignment focuses on all of what we had learned so far, the process, group work, design development, storytelling, interactivity, usability, coding and presentation.

My Role

My part of the assignment was making the mood board and the storyboard, drawing the characters for the game(5). Making a front page, a sprite animation, gif, javascript for the stars on front page and in the end all of the team put everything together and made sure it was going good. Me and Astrid made a facebook page and put in our gif advertisement.

What I learned

I found myself in a well put together group where we had different talents. The flow was well organised and everybody had their role. I found that Kanban chart is very useful and helpful and the storyboard is a key. I got a bit more insights into javascript and cpuld understand the game code more than I thought I would.
